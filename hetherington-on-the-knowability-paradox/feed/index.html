]]> By: hetherington http://certaindoubts.com/hetherington-on-the-knowability-paradox/#comment-3815 Sun, 20 Aug 2006 21:49:41 +0000 http://fleetwood.baylor.edu/certain_doubts/?p=608#comment-3815 I’m following these comments with great interest (because they are substantial and intriguing in their own right). But I’m not sure that all of them bear upon the strictly epistemic dimension of my argument’s focus.

That focus is upon knowledge AS AGAINST mere true belief. Is there a paradox of knowability AS AGAINST a paradox of mere true belief? Even if we accept (as I don’t; see entry #6 above) that knowledge must include true belief, that question is reasonable. There is quite a difference between these two theses: (1) Any truth is believable; (2) Any truth is not only believable, but justifiably believable. And insofar as (again for simplicity, I’m assuming here; but again, see entry #6 above) knowledge is NOT merely true belief, the following obtains: If there is no paradox of (2)-knowability, then there is no knowability paradox about knowledge as such. Or (to state the latter consequent more helpfully) there is no knowability paradox about knowledge in any sense that touches upon what epistemologists routinely ask of, or expect from, knowledge qua something BEYOND mere true belief. There would be no knowability paradox about the strictly epistemic dimension of knowledge.
